About git log
-------
参考网址：https://www.atlassian.com/git/tutorials/git-log/
<br/>
<br/>
####发现最近的三个commit<br/>
```
   git log -3
```
<br/>
####某时间段内的commit<br/>
```
   git log --after="2014-7-1" --before="2014-7-4"
```
<br/>
####查找某人的commit<br/>
```
   git log --author="John"
   git log --committer="John\|Mary"
```
<br/>
####查找最近的三个commit<br/>
```
   git log -3
```
